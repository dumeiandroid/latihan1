Cara Penggunaan Update:

1. Akses halaman dengan parameter URL:

update_kolom.html?id=123&table=latihan

2. Data akan dimuat otomatis berdasarkan ID

3. Edit data yang diperlukan

4. Klik tombol "Update" untuk menyimpan perubahan


Custom Columns:

Gunakan atribut data-columns untuk menentukan kolom yang diinginkan:

data-columns="x_01,x_05,x_10"

Jika tidak ada data-columns, akan menggunakan urutan default: x_01, x_02, x_03, dst.
