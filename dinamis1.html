Cara Penggunaan

Untuk menggunakan Dynamic Form Handler, Anda perlu:

  1. Pastikan file tambah.js sudah di-load di halaman
  2. Tambahkan atribut pada form sesuai kebutuhan
  3. Sesuaikan konfigurasi API di file JavaScript

Atribut Form yang Tersedia:

data-api-enabled="true" - Mengaktifkan form handler data-table="nama_tabel" - Menentukan tabel tujuan data-reset="true/false" - Reset form setelah sukses (default: true) data-show-alert="true/false" - Tampilkan pesan alert (default: true) data-redirect="/url" - Redirect setelah sukses data-method="POST/PUT" - HTTP method (default: POST) data-update-id="123" - ID untuk update (otomatis jadi PUT)

Konfigurasi JavaScript yang Perlu Disesuaikan:

const dynamicFormHandler = new DynamicFormHandler({ apiBaseUrl: '/api/contacts', // Sesuaikan dengan URL API Anda defaultTable: 'contacts', // Tabel default maxColumns: 20, // Maksimal kolom (x_01 - x_20) debug: true // Set true untuk development });
