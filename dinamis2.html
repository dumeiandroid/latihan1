Dynamic Form Handler - Contoh Penggunaan

Form untuk Tabel "latihan"

Atribut: data-api-enabled="true" data-table="latihan" data-reset="true"

Form untuk Tabel "contacts" (Default)

Atribut: data-api-enabled="true" (menggunakan tabel default)

Form dengan Redirect

Atribut: data-table="latihan" data-reset="false" data-redirect="/success.html"

Form untuk Update Data

Atribut: data-table="latihan" data-update-id="123" (akan menggunakan method PUT)

Form dengan Banyak Field (Test Maksimal Kolom)

Atribut: data-table="latihan" (akan mapping ke x_01 sampai x_20)

Cara Penggunaan

Untuk menggunakan Dynamic Form Handler, Anda perlu:

  1. Include file JavaScript: <script src="tambah.js"></script>
  2. Tambahkan atribut pada form sesuai kebutuhan
  3. Sesuaikan konfigurasi API di file JavaScript

Atribut Form yang Tersedia:

data-api-enabled="true" - Mengaktifkan form handler data-table="nama_tabel" - Menentukan tabel tujuan data-reset="true/false" - Reset form setelah sukses (default: true) data-show-alert="true/false" - Tampilkan pesan alert (default: true) data-redirect="/url" - Redirect setelah sukses data-method="POST/PUT" - HTTP method (default: POST) data-update-id="123" - ID untuk update (otomatis jadi PUT)

Include JavaScript:

<script src="tambah.js"></script>

Contoh Form Sederhana:

<form data-api-enabled="true" data-table="latihan"> <input type="text" name="nama" placeholder="Nama" required> <input type="email" name="email" placeholder="Email" required> <textarea name="pesan" placeholder="Pesan"></textarea> <button type="submit">Simpan</button> </form>
